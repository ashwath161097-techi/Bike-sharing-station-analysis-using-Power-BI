# Bike Station Data Analytics Dashboard (Power BI)

## Project Overview

This project analyzes bike station data to monitor:

- Bike availability
- Station utilization
- Capacity vs availability
- Banking facility availability
- Station performance

The dashboard helps improve operational efficiency and customer satisfaction.

---

## Tools Used

- Power BI
- DAX
- Data Cleaning
- Data Modeling
- Data Visualization

---

## Key Features

- Map visualization of stations
- Capacity vs availability comparison
- Top 10 stations analysis
- Banking facility analysis
- Station status monitoring

---

## Files Included

- Power BI Dashboard (.pbix)
- Project Report (.pdf)
- Documentation (.docx)

---

## Key Insights

- Uneven distribution of bikes across cities
- Some stations have low availability
- Banking-enabled stations improve usability
- Capacity underutilization in some areas

---

## Recommendations

- Implement smart redistribution strategy
- Expand banking facilities
- Optimize capacity allocation
- Improve station uptime
